Make Address Argument.

Factory method. Make address argument from program memory, under given offset

Note
offset is input/output parameter. As input it contains offset in program memory. It is updated by the function to and contains value bits of the argument
Parameters
programMemoryReference to program memory
offsetAs input - offset to first bit of the argument. As output - offset to a bit after last bit of the argument
