OperationPtr Evm::Operation::makeOperation ( const Utils::BitBuffer &  programMemory,
uint32_t &  offset 
)

Opcode decoder.

Parse program memory from given offset and determine operation from opcode.

Note
Offset is input/output reference. After execution it points to the next bit after argument set, that has been required by the current instruction
Parameters
programMemoryReference to program memory
offsetInput/output reference. As input, offset to first bit of first argument for current instruction. As output - offset to first bit after the last argument required by the instruction
Returns
Object of IOperation interface
